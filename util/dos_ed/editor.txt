존 에디트 사용 설명서.

일단, 존 에디트에서 사용하는 한글은 완성형 한글을 사용합니다.
에디트 자체는 한글이 없으므로, HBIOS 등의 한글 에뮬레이터를 사용해야
합니다. 물론 완성형으로 띄워야 합니다.

일단 editor 를 실행하기 전에 디렉토리를 만들어줘야 합니다.
editor.exe 화일이 있는 디렉토리에 서브디렉토리로 ₩rooms 디렉토리를
만들어줘야 합니다. 그곳에 room의 데이타가 저장되며, 서브디렉토리가
없다면 화일을 읽을수가 없습니다.

한글 에뮬레이터를 띄우고, editor.exe 화일을 실행시키면.
제일 먼저 Room 의 번호를 적으라고 나옵니다.
여기에서 자신이 만들고자 하는 방의 방번호를 입력하면 다음과 같은 메뉴
가 나옵니다.

 1. Edit name
 2. Edit short description
 3. Edit long description
 4. Edit level/trap/track/special
 5. Edit flags
 6. Edit random monsters
 7. Edit permanent monsters
 8. Edit permanent objects
 9. Edit room statistics
10. Edit exits
11. Clear this room
12. Abort this room
13. Back to main menu

각 메뉴를 설명하면,
1번 메뉴는 그 방의 이름을 정해주는 것입니다.
2번 메뉴는 그 방의 짧은 설명을 적어주는 것인데, 존 제작에 있어서는
    이 메뉴는 사용하지 않습니다. (짧은 설명을 사용하지 않기 때문)
3번 메뉴는 그 방의 긴 설명을 적어주는 것으로, 머드에서 방의 설명부분이
    바로 이 메뉴로 적어주는 것입니다.
4번 메뉴에 들어가면, 서브 메뉴로..
       Trap    Trap exit   special
       Track string
       Low level
       High level
 Trap은 함정을 설정해주는 것으로 1 - 7 번까지 설정이 가능합니다.
       1 : 보통의 일반적 함정 (체력이 떨어지는)
       2 : Poison dark trap (뭔지 저도 잘 모름.. ^^;)
       3 : 돌이 떨어지는 함정 (선녀에게 갈때 나오는 트랩)
       4 : 체력이 떨어지지 않고 도력이 떨어짐..
       5 : Spell loss trap (주문을 잃는다는 뜻인데... --;)
       6 : 플래이어가 가진 모든 아이템을 잃어버리는 함정
       7 : monster alarm trap (해석이 안됨.. --;)
 Trap exit 는 함정에 빠졌을때 가는 방을 뜻하는 것입니다.
 special 은.. 저도 정확하게는 모르지만, 존제작에 있어서는 필요없습니다.
 Track string 은 포졸의 '추적'을 사용했을때 나오는 흔적을 뜻하는데,
      물론 존제작에 있어서는 필요없습니다.
 low/high level 은 그 방에 들어갈수 있는 플레이어의 레벨 제한입니다.
5번 메뉴는 그 방의 플래그를 설정해주는 것입니다.
    플래그는 같이 있는 flag.doc 화일을 참고해 주시기 바랍니다.
6번 메뉴는 그 방에 랜덤하게 나오는 몹을 정해주는 것입니다.
    존제작 단계에서는 할 필요가 없습니다.
7번 메뉴는 그 방에 고정적으로 나오는 몹을 정해주는 것입니다.
    물론 존제작 단계에서는 할 필요가 없습니다.
8번 메뉴는 그 방에 고정적으로 나오는 아이템을 정해주는 것입니다.
    물론 존제작 단계에서는 할 필요가 없습니다.
9번 메뉴는 저도 뭔지 모르겠군요. 존제작에는 필요가 없는 것입니다.
10번 메뉴는 그 방의 출구를 만드는 것입니다. 서브 메뉴로는
   1. Add exit   2. Modify exit  3. Delete exit
   4. List exit  5. Back to room menu
   1번 메뉴는 새로운 출구를 만드는 것으로, 선택하게 되면
     Name (출구의 이름), Room (연결되는 방의 방번호)
     Key (만약 문이 잠겨져 있을 경우에 열쇠의 번호)
     Time Interval, Last Time, Misc 의 세가지 메뉴는 그냥 엔터를 치세요.
     아무 쓸모 없는 거니까요..
     Flag #: 이것은 exit의 플래그를 정해주는 것입니다.
     exit flag 는 같이 있는 flag.doc 화일을 참조하십시오.
   2번 메뉴는 이미 만들어둔 출구를 수정할때 사용합니다.
   3번 메뉴는 이미 만든 출구를 삭제할때 사용합니다.
   4번 메뉴는 만들어둔 출구의 리스트를 보는 것입니다.
   5번 메뉴는 주 메뉴로 돌아갑니다.
11번 메뉴는 그 방의 설정된 내용을 모두 삭제하는 것입니다.
12번 메뉴는 지금까지 작업한 내용을 저장하지 않고 도스상으로 나갑니다.
13번 메뉴는 모든걸 저장하고 도스상으로 나갑니다.

존제작에 있어서는 1, 3, 5, 10 번 메뉴만 사용하고..
필요시(trap을 설정했을때)에만 4번 메뉴도 사용하시기 바랍니다.
다른 메뉴는 사용하게 되면 나중에 머드에 띄울때 엉망이 될 가능성도 있으
니 사용하시지 말기를 바랍니다.
